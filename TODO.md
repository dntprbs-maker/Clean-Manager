# 📋 작업해야 할 항목 (Backlog)

앞으로 실제 앱 환경 구축을 위해 진행해야 할 작업 목록입니다. 사장님께서 특정 작업을 지시하시고 완료될 때마다 이 문서에서 해당 항목은 삭제해 나가겠습니다.

- [ ] **데이터베이스(DB) 연동 및 실시간 동기화**
  - 파이어베이스(Firebase) 또는 수파베이스(Supabase) 클라우드 DB 연결
  - 사장님이 일정을 등록하면 모든 직원의 기기에서 실시간으로 동기화되도록 구현
  - 현재 브라우저 저장소(LocalStorage)에 의존하는 구조 탈피

- [ ] **실제 로그인 및 권한(보안) 시스템 도입**
  - 카카오톡 간편 로그인 또는 이메일/비밀번호 로그인 구현
  - 실제 계정 기반으로 접근 권한(최고관리자, 팀장, 팀원) 완벽 분리
  - 일반 직원은 절대 타인의 권한이나 관리자 화면에 접근할 수 없도록 보안 강화

- [ ] **공지사항 & 외부 링크 실제 데이터 연동**
  - 부가 기능 화면(공지사항, 링크, 작업 내역 등)의 임시 데이터를 실제 DB에 영구 저장되도록 연동
  - 등록, 수정, 삭제(CRUD) 기능의 백엔드 연동 완성

- [ ] **푸시 알림 기능 도입**
  - 새로운 일정 등록, 기존 일정 수정 및 삭제 시 해당 팀원들에게 스마트폰 푸시 알림 발송

- [ ] **사진 및 첨부파일 기능 (현장 업무용)**
  - 스케줄 상세 화면에서 작업 전/후 현장 사진을 업로드하고 언제든 조회할 수 있는 기능 추가
