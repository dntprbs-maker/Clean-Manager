# 📋 작업해야 할 항목 (Backlog)

앞으로 실제 앱 환경 구축을 위해 진행해야 할 작업 목록입니다.

---

## 🔥 1순위 — 오늘 저녁 작업

- [ ] **Firebase 프로젝트 연동 (DB + Storage + 인증 전체 세팅)**
  - firebase.google.com 에서 프로젝트 확인 또는 신규 생성
  - Firestore DB 활성화 → 일정/완료보고/직원 데이터 저장
  - Firebase Storage 활성화 → 현장 사진 업로드 연동
  - 환경변수(.env) 설정 후 GitHub Secrets 등록
  - 현재 LocalStorage 의존 구조 전면 교체

---

## 📌 2순위 — Firebase 연동 후 바로 이어서

- [ ] **현장 완료 보고 실제 기능 연결** (현재 목업 상태)
  - 청소 전/후 사진 → Firebase Storage 실제 업로드
  - 완료 보고 데이터 → Firestore 저장 및 사장님 대시보드 반영
  - AI 워크플로우 로그 → 실제 저장 및 조회 가능하도록

- [ ] **실제 로그인 및 권한(보안) 시스템 도입**
  - 카카오톡 간편 로그인 또는 이메일/비밀번호 로그인
  - 실제 계정 기반으로 권한(최고관리자, 팀장, 팀원) 완벽 분리
  - 현재 드로어 내 계정 전환 방식 제거

- [ ] **공지사항 & 외부 링크 실제 데이터 연동**
  - 임시 데이터 → Firestore 영구 저장
  - 등록·수정·삭제(CRUD) 실제 연동 완성

## 🎨 목업 완성 후 실제 연결 필요한 화면 목록

현재 UI는 만들어져 있지만 실제 동작이 안 되는 화면들:

- [ ] **현장 완료 보고** — 사진 업로드, 완료 데이터 저장
- [ ] **일정 요약(대시보드)** — 실제 매출/건수 집계 연동
- [ ] **팀 공지사항** — 새 공지 작성·저장·삭제
- [ ] **최근 작업 내역** — 앱 껐다 켜도 유지되도록 DB 저장
- [ ] **외부 링크** — 추가한 링크 DB 저장
- [ ] **직원 관리** — 직원 정보 DB 저장 (현재 앱 껐다 켜면 초기화)
