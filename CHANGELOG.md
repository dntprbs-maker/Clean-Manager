# 📝 클린메니저 (Clean-Manager) 작업 일지

이 파일은 지금까지 작업한 내용과 앞으로 업데이트될 내역을 상세히 기록하는 파일입니다. 
새로운 기능이 추가되거나 수정될 때마다 아래에 내역이 누적됩니다.

---

## [v1.2.0] - 2026-06-19 (현재)
### ✨ 신규 기능 (Features)
- **팀 관리(추가/삭제) 기능 구현**
  - 기존에 소스코드에 고정되어 있던 팀 목록을 동적으로 관리하도록 구조 개편
  - `[직원 관리]` 화면에 `[팀 관리]` 버튼 추가 및 전용 관리 모달(창) 구현
  - 새로운 팀을 자유롭게 추가 가능 (추가 시 직원 필터, 폼, 팀별 일정 등에 즉시 연동)
  - 팀 삭제 기능 지원: 실수로 지우는 것을 방지하기 위한 확인창 제공
  - **직원 자동 소속 변경**: 팀 삭제 시 해당 팀에 소속되어 있던 팀장/팀원의 소속이 자동으로 '미정'으로 일괄 변경되도록 안전장치 구현

### ♻️ 구조 및 UI 개선 (Refactoring & UI)
- **사이드 메뉴(SideDrawer) 구조 전면 개편**
  - 메뉴 중간에 있던 '부가 기능' 텍스트 타이틀 삭제
  - 프로필 우측 상단에 있던 `[직원 관리]` 버튼을 전체 메뉴 리스트 최상단으로 이동 배치 (사장, 관리팀 전용)
  - `[직원 관리]`와 `[일정 요약]` 버튼 사이에 `[팀별 일정]` 메뉴 신규 추가
- **팀별 일정 화면 추가**
  - 현재 화면에 표시되는 일정들을 팀별로 몇 개씩인지 집계하여 리스트업해 주는 전용 화면 추가

### 🐛 버그 수정 (Bug Fixes)
- **일정 요약(Dashboard) 화이트 스크린 크래시 해결**
  - 날짜 변환 함수명을 잘못 호출(`fmtDate` -> `fmt`)하여 발생하던 리액트 렌더링 중단 에러 수정

## [v1.1.0] - 2026-06-18
### ✨ 신규 기능 (Features)
- **직원 관리 및 권한 제어 시스템 도입**
  - 가상의 계정 시스템 추가 (최고관리자, 팀장, 팀원) 및 로그인 전환 기능 (테스트용)
  - `사장`, `관리팀`만 열람 및 조작 가능한 `[직원 관리]` 전용 화면 개발
  - 직원 정보(이름, 연락처, 소속팀, 직급) 등록 및 수정 기능
- **4가지 부가 기능 화면 도입**
  1. **일정 요약 (대시보드)**: 오늘/내일 일정 개수 집계 위젯
  2. **팀 공지사항**: 전사 및 팀별 공지사항 게시 (관리자만 작성 가능)
  3. **최근 작업 내역 (로그)**: 캘린더 등록, 수정, 삭제 시 발생한 작업 내역을 실시간 타임라인으로 기록
  4. **자주 쓰는 외부 링크**: 원하는 URL(예: 네이버 지도, 카카오 채널 등)을 사용자가 직접 추가하고 새 창으로 열 수 있는 기능

### ♻️ 구조 및 UI 개선 (Refactoring & UI)
- **세부 일정 보기 UI 변경**
  - 기존에 아래에서 절반만 올라오던 바텀시트(Bottom Sheet) 형태에서, 뒷배경 없이 꽉 차는 **전체화면(Full Screen)** 형태로 전환
- **캘린더 팀 선택 필터 개선**
  - 캘린더 상단에 배치된 팀 필터 칩들이, 화면을 아래로 스크롤하더라도 상단에 딱 붙어서(Sticky) 가려지지 않도록 고정 처리 완료

## [v1.0.0] - 2026-06-18 (프로젝트 초기 세팅)
### 🎉 초기 세팅 및 핵심 기능
- **Naver Calendar UI 완벽 구현**
  - `React` + `Vite` + `Tailwind CSS` 기반의 모바일 친화적 웹앱 구축
  - 위아래 스크롤을 통해 3단계 뷰(Full Month ↔ Half Sheet ↔ List Only)로 자연스럽게 전환되는 복합 스와이프 인터페이스 엔진 자체 구현
- **기본 캘린더 기능**
  - 일정(이벤트) 등록, 수정, 삭제 폼 구현
  - 휴일(공휴일) 데이터 렌더링
  - 캘린더 검색 기능 및 하단 플로팅 버튼(FAB) 추가
- **배포 연동**
  - `GitHub Pages` (GitHub Actions) 연동을 통해 커밋 시 자동 웹 배포되는 CI/CD 환경 구성 완성
